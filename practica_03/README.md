Práctica 3: Modelación con GLM
========================

Esta práctica tiene como objetivo que los alumnos apliquen los conceptos de modelación con GLM utilizando un conjunto de datos simulado. Se proporcionan ejemplos de código en Python/R para guiar a los estudiantes en el proceso de análisis y modelación.

El cuaderno `GLM_modelacion.ipynb` contiene ejemplos de cómo ajustar modelos de frecuencia y severidad utilizando GLM, así como cómo interpretar los resultados y realizar visualizaciones. Al final del cuaderno se añadieron 5 preguntas sencillas que requieren pequeños cambios de código, siguiendo la misma lógica de los ejemplos anteriores.

El alumno debera responder las preguntas al final del cuaderno, modificando el código proporcionado para adaptarlo a las nuevas variables o escenarios planteados.

Además, se incluyen los scripts `practica_glm_auto_studentpy` y `practica_glm_auto_student.R` que contiene un conjunto de preguntas con intrucciones para que los alumnos las resuelvan utilizando Python o R, respectivamente. El objetivo es que los estudidantes puedan crear el código por sí mismos, basándose en las instrucciones y ejemplos proporcionados en cada script.
